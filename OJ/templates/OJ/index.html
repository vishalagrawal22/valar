Problem statement

You are given an array of length N, and you will be asked to answer Q queries. Each query will contain 2 values, L and R, and you have to output the sum of the elements between the Lth and Rth elements (inclusive) of the array.

testcase format:

  • first line contains an integer N, the number of elements in the array
  • next lines contains N integers, the elements in the array
  • third line contains an integer Q, the number of queries
  • each of the following Q lines contain 2 integers L and R, the boundaries of the query

constraints:

  • N <= 10^5
  • all elements individually <= 10^4
  • Q <= 10^5
  • L <= N
  • R >= L

sample IO:

Input:

5

1 14 2 67 100

3

2 2

2 5

1 4

Output:

14 183 84
